Plugin para gerenciar uploads e atualizações em massa, incluindo textos alternativos de imagens, meta tags do Yoast SEO, exportação de posts e categorias.

== Description ==

O Ferramentas Upload é um plugin WordPress que oferece funcionalidades para gerenciamento em massa de diversos aspectos do seu site:

* Atualização em massa de textos alternativos (alt text) de imagens
* Atualização em massa de meta tags do Yoast SEO
* Exportação de posts e categorias
* Movimentação em massa de posts para a lixeira

= Principais Funcionalidades =

* Atualização de textos alternativos via CSV
* Atualização de meta tags do Yoast SEO via CSV
* Exportação de posts com filtros personalizados
* Gerenciamento em massa de posts

== Installation ==

1. Faça o upload dos arquivos do plugin para a pasta `/wp-content/plugins/ferramentas-upload`
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Acesse as funcionalidades do plugin através do menu 'Ferramentas Upload' no painel administrativo

== Frequently Asked Questions ==

= O plugin é compatível com o Yoast SEO? =

Sim, o plugin é totalmente compatível com o Yoast SEO para atualização de meta tags.

= Como faço para atualizar os textos alternativos em massa? =

Prepare um arquivo CSV com duas colunas: URL da imagem e texto alternativo desejado. Depois, acesse a aba 'Alt Text' no menu do plugin e faça o upload do arquivo.
